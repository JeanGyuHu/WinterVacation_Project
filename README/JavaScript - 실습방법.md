# [JavaScript - 실습방법](https://opentutorials.org/course/1375/6639)

> 크롬 웹브라우저와 Sublime text 를 이용

- 크롬 개발자 도구
  - Elements : HTML의 태그를 보여준다
  - Console : 즉각적으로 코딩할 수 있음
- Sublime text3
  - html + TAB
  - script + TAB

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>
    <script type="text/javascript">
      alert(1);
    </script>
    
  </body>
</html>
```
