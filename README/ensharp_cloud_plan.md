# ENSHARP CLOUD

- 엔샵 동아리 내에서 자료를 관리하기 위해 제작할 웹 클라우드 서비스 입니다.

- 해당 기수가 활동했던 자료들이 시간이 지남에 따라 뒤섞이고, 관리가 제대로 되지 않는다는 점에 착안하였습니다.

- 각 기수마다의 자료를 그때그때 보관한다면 추후의 특정 기수뿐만 아니라 몇 기수 건너서 자료를 참고할 수 있도록 제공할 예정입니다

  - #### 프로젝트에 필요한 것

    - **HTML, CSS**
      - **기본 레이아웃, 디자인**
      - **반응형 웹**
        - 모바일 지원
    - JS, Server, DB
      - 파일관리
        - **다운로드, 업로드**
          - **Drag & Drop**
        - 공유
          - URL
        - 검색 (알고리즘)
        - 미리보기(썸네일)
        - 뷰어 연동 (?)
      - 회원관리 (Server)
        - **로그인**
        - **회원정보 및 등급**
      - 권한
        - 파일 및 폴더에 대한 접근권한
      - 디렉토리 구조
        - 스크롤
        - 정렬 (알고리즘)
        - 이름, 생성일, 저자
